#Go_Code

@快捷键风格设置：
File->Settings->Keymap->选择Eclipse风格->确定

@基于Eclipse风格的常用快捷键：
·tab：缩进

·shift + tab：反缩进

·快速复制: ctrl + alt + ↓

·删除当前行: ctrl + d

·格式化代码：ctrl + alt + l（L）

·快速注释/解注释：ctrl + /

·快速移动选区：alt + ↑或↓

·重命名变量或函数：alt + shift + r

·撤销上一步操作：ctrl + z

·解撤销：ctrl + y

·快速生成解决方案：alt + Enter

·快速查找：ctrl + F

·将光标切回上次位置：alt + left/right

·折叠所有：ctrl + shift + 小键盘/

·展开所有：ctrl + 小键盘*

·切换大小写：ctrl + shift + y


·快速包裹：ctrl + alt + t
·快速抽取选区为参数：ctrl + alt + p
·快速插入覆写方法：alt + insert
